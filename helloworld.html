各种编程语言的Hello World示例

Python

print("Hello, World!")

C

#include <stdio.h>
int main() {
  printf("Hello, World!\n");
  return 0;
}

C++

#include <iostream>
int main() {
  std::cout << "Hello, World!" << std::endl;
  return 0;
}

Java

public class HelloWorld {
  public static void main(String[] args) {
    System.out.println("Hello, World!");
  }
}
